# Tic-Tac-Toe AI

Консольная игра крестики-нолики с возможностью играть против компьютера (AI).

## О проекте
Эта игра позволяет игроку играть в крестики-нолики против простого AI.  
Игра работает в консоли и подходит для новичков, кто хочет весело провести время или изучить основы Python.

## Функциональность
- Играть против компьютера (AI)  
- Проверка победителя после каждого хода  
- Определение ничьи, если все клетки заняты  
- Возможность начинать новую игру после завершения

## Требования / зависимости
- Python 3.10+  
- Библиотеки: нет (все стандартные модули Python)
